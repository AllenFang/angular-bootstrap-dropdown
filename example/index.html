Tutorial



        1. Use bs-dropdown-display attribute in bsDropdown to specify the default text on dropdown

        2. Use bs-dropdown-items(required) attribute in bsDropdown to specify the options you want to display in dropdown

        3. Use ng-model(required) attribute in bsDropdown to specify the current selected option in dropndown.
 
           If no data you want to select on doopwodn default, you can give null value.
 
           For example, you can modify $scope.selectYear = null; in YearController in this example.

        4. Use ng-change to listen a change event on bsDropdown.
 

        5. Use bs-dropdown-divider to specify the divider, for example bs-dropdown-divider="{{[2,5]}}".
  

        6. Use bs-dropdown-item-disabled to specify which option should be disabled,

           for example bs-dropdown-item-disabled="{{[2,5]}}".
  

        7. Use bs-dropdown-disabled to set bsDropdown disabled, for example bs-dropdown-disabled="true".
